# 🏀 CRUD de Times da NBA

## 📌 Sobre o Projeto
Este é um sistema **CRUD (Create, Read, Update, Delete)** para gerenciamento de times da NBA. O projeto foi desenvolvido em **Java** com **MySQL**, permitindo cadastrar, visualizar, atualizar e excluir informações sobre times de basquete.

## 🛠 Tecnologias Utilizadas
- **Java** (POO)
- **MySQL** (Banco de Dados)
- **JDBC** (Conexão com o banco)
- **Git/GitHub** (Controle de versão)

## 🚀 Funcionalidades
✔ **Cadastrar** um novo time no banco de dados
✔ **Listar** todos os times cadastrados
✔ **Atualizar** informações de um time existente
✔ **Excluir** times do sistema

## 🎯 Como Executar o Projeto
### 📌 Pré-requisitos
Antes de começar, certifique-se de ter os seguintes softwares instalados:
- Java (JDK 8+)
- MySQL
- Um editor de código (Eclipse, IntelliJ IDEA, VS Code, etc.)
